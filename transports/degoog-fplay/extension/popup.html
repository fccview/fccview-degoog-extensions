Degoog - fplay connector
Degoog fplay connector, inspired by 4get.

First and foremost, credits where credits is due. This idea came from the lovely lolcat, creator of 4get. After a nice chat in the degoog discord server I decided it was worth creating a transport for it.

Use the same port and optional password as in Degoog -> Transports -> Fplay -> Configure. You can also toggle “strip engine cookies / user agents” to force cookies and agents via fplay.
